### Installation
___
It's just like any other Simple project. Basically here is how you use it for yourself.

.

- Clone the repo `https://github.com/aminesmkhani/event-app-vue`
- `cd` to project folder.
- Run `npm install`
- After `npm run serve`
- Open Browser `localhost:8080`
- Done !!! Enjoy `Run` app
